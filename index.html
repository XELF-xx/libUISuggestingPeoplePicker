libUISuggestingPeoplePicker by XELF

A iPhone library providing an UIView, which acts like the contact picker of the built-in SMS app

libUISuggestingPeoplePicker is a library for iPhone development. You can include it in your own project to present a view using which your user can add multiple phones by editing with numberpad or picking them from contacts, just like the built-in SMS app in iPhone.
